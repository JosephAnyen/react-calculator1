# Calculator App

A simple, responsive calculator built with React and TypeScript.  
It supports basic arithmetic operations with a clean and intuitive UI.

## Features

- Basic arithmetic operations (+, −, ×, ÷)
- Clear (AC) and delete (DEL) buttons
- Responsive layout using CSS Grid
- Built with modern React and TypeScript

## Tech Stack

- React
- TypeScript
- Vite
- CSS Grid

## Getting Started

### Install dependencies

```bash
npm install
npm run dev


```
